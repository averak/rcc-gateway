## Develop

### Requirements

- Node.js 14
- Angular

### Installation

```sh
$ yarn
```

### How to build

```sh
$ yarn build
```

When build successful, you can find static files in `dist`.

### How to run

```sh
$ yarn start
```

### How to check code

You need to do this before committing.

```sh
# lint & type check
$ yarn run check

# code format
$ yarn run format

# test
$ yarn run test
```
