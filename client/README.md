This app is front-end of RCC Gateway.

## Develop

### Requirements

- Node.js 14
- yarn

### Installation

```sh
$ yarn
```

### How to build

```sh
# local build
$ yarn build

# production build
$ yarn run build --configuration production
```

### How to run dev-server

```sh
$ yarn start
```

Now, you can access to [localhost:4200](http://localhost:4200/).

### How to check code

You need to do this before committing.

```sh
# lint & type check
$ yarn run check

# code format
$ yarn run format

# test
$ yarn run test
```
